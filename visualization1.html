Visualization 1

Interpretation

This is example of a bar chart that uses data from police districts and aggregate the total of each district. Each bar will represent a district and the height is the number of incidents a district has. I used a d3 map. It used the key as the district and the count as the value.

Discussion

The primary goal is to get each police district and see the differences between districts. This can be helpful to a user to determine the crime rate in a police district in order to possibly add more force to those areas. This was the first chart I created, and I found that there was the learning curve hump that I have passed.
